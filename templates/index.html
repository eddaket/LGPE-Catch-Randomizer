Let's Go Catch Randomizer

Welcome to the Let's Go Catch Randomizer! This tool can be used to generate a Tracker File containing 50 random, obtainable Pokémon. The intent is for you, the player, to use only those 50 Pokémon to play through the Pokémon Let's Go games!

Settings

Use this to restrict the amount of 1% encounters (e.g. Kangaskhan).
